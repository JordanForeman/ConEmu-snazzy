# conemu-snazzy

> ConEmu theme with bright colors

## Install

Copy the contents of `snazzy.xml` file and paste them into your `ConEmu.xml` file in the correct location. For reference: https://github.com/joonro/ConEmu-Color-Themes#manually-add-color-schemes-and-modify-the-conemuxml-file.
